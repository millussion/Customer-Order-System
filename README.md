# Customer-Order-System

## Project Description 📌

_This project is a simple Customer Order Management System developed in Python.
It allows users to register customers, manage products, create orders, and generate sales reports._

_The main purpose of this system is to replace manual processes (like spreadsheets or paper records) with a more structured and automated solution._

---

## System Features ⚙️

_The system includes the following functionalities:_

  * Register customers (ID, name, email)
  * Register products (ID, name, price)
  *  Create orders (customer + product + quantity)
  * View registered orders
  * Calculate total daily income
  * Generate a final report with sales summary


---

## Technical Approach 🧠 

 _This project was built using:_

  * Dictionaries → to store customers and orders
  * Tuples → to represent product data (immutable structure)
---

## Code Structure 🧩

_The system is modular and uses reusable functions.
Each function:_

*  Receives parameters
*  Returns values
*  Avoids direct printing inside logic
--- 

## System Output 📊

_At the end of execution, the system generates:_

 * Total number of orders
 * Total income
 * Orders grouped by customer
 * List of products sold
